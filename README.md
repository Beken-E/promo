# Система Промокодов (Promo Demo)

Демонстрационный проект на **NestJS**, реализующий систему управления промокодами, регистрацию пользователей с бонусами и отчетность.

## 🚀 Основные возможности

*   **Регистрация пользователей**: Создание аккаунта с автоматическим открытием счета.
*   **Система промокодов**: 
    *   Создание кодов с процентом скидки.
    *   Лимиты на количество активаций.
    *   Проверка срока действия.
    *   Гарантия "один код в одни руки" (привязка к email).
*   **Бонусная программа**: Автоматическое начисление средств на баланс при регистрации с валидным промокодом.
*   **Отчетность**: Эндпоинт для получения списка всех пользователей, их балансов и истории активаций.

## 🛠 Технологический стек

*   **Backend**: NestJS (v11), TypeScript
*   **Database**: PostgreSQL
*   **ORM**: TypeORM
*   **Security**: Bcrypt (хэширование паролей)
*   **Documentation**: Swagger UI
*   **Infrastructure**: Docker Compose

## 🏁 Быстрый старт

### 1. Подготовка окружения
Убедитесь, что у вас установлены Node.js и Docker. И задать корректные значения для env файла

### 2. Запуск базы данных
```bash
docker-compose up -d
```

### 3. Установка зависимостей и запуск приложения
```bash
npm install
npm run start:dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

## 📖 API Документация (Swagger)

Интерактивная документация доступна по адресу:
[http://localhost:3000/api](http://localhost:3000/api)

## 📋 Основные Эндпоинты

### Авторизация и Регистрация (`/auth`)
*   `POST /auth/register` — Регистрация нового пользователя.
    *   Принимает: `email`, `login`, `password`, `promoCode` (опционально).
    *   Если промокод валиден, на счет пользователя начисляется бонус.

### Пользователи и Отчеты (`/users`)
*   `GET /users/report` — Полный отчет: список всех пользователей, их текущий баланс и история активированных промокодов.

### Промокоды (`/promo-codes`)
*   `POST /promo-codes` — Создание нового промокода (установка лимитов и срока действия).
*   `GET /promo-codes` — Список всех созданных кодов и статистика их использования.
*   `GET /promo-codes/:code` — Детальная информация по конкретному коду.
